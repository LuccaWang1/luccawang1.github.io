How I Built This Website

  1. Authorship: First off, I hand coded each character of this site. The code came from me, Lucca Wang -- not the ever popular ChatGPT or somewhere else.

  2. Start: I engaged in strategic planning and conducted research. I thought, "What information do I need to include on this portfolio website?"

  3. Stack and technologies: To quickly launch my portfolio, I used HTML and CSS with VSCode and deployed it via Github Pages. As I developed the site, I explored and implemented solutions within this setup. I love using Git version control to manage the development process.

  4. Domain: I selected a domain provider and registered a domain name, then configured it by updating the CNAME file.

  5. Asset gathering: I gathered and organized my assets, including a PDF file and various images - primarily PNG files, as they are well-suited for web use. I took care with naming conventions and the directory structure. I customized images via icons8 for the favicon and for the social media/contact icons.

  6. Webpages: I built out index (the homepage) with Hypertext Markup Language (HTML), then added this webpage. I've refined my HTML coding style to be both elegant and readable. I'm a fan of the alt and title attributes and section tags to enchance accessibility.

  7. Styling: I ensured the website was consistent and accessible across both computer monitors and mobile screens by implementing responsive design with CSS media queries. I adhered to ADA guidelines for font size and applied strong design principles, such as typographic hierarchy. I enjoyed working with CSS selectors, especially leveraging classes and DOM elements. The animations add a dynamic touch.

  8. Learning: Through the process of building the website, I gained insights into both my personal strengths and technical skills. I deepened my understanding of CSS and also self-taught about Open Graph to ensure information displays correctly on social media platforms when the portfolio URL is shared. I expanded my knowledge on not only everything stated on this webpage regarding how I built the website but also my knowledge on the website's content itself.

  9. Gratitude: I'm grateful I decided to build a developer portfolio website for myself, for the work and growing involved in developing it, and for you visiting!
