About

I'm Lucca Wang, a full-stack software engineer with a relentless drive to learn and grow.

In my recent role as a full-stack developer in Enterprise (2024-2025), I developed software in Python, Django, React, and our old friend HTML, contributing to the full-stack (both frontend and backend repositories). I improved code maintainability, user experience, and system performance.

I also gained experience with Docker, AWS ECS, EC2, and IAM, along with tools and libraries such as Postman, MySQL, and Material UI. I developed the app's first unprotected URLs/public-facing and branded web pages that work with the backend and the external Stripe API, testing with web hook calls and the Django interface in the Development environment. I implemented responsive design and the WCAG accessibility standards.

Read about the issue ticket I helped resolve for application security and centralized configuration management, or more about my recent professional experience as full-stack developer in Enterprise, on my blog.

Some of my personal projects are on this webpage below. In addition to Python, React, and HTML, I developed these apps with Flask, Python templating engine Jinja2, JavaScript, CSS, PostgreSQL, ORM SQLAlchemy, Bootstrap, and the Cloudinary, Google Fonts, and Full Calendar APIs. I gained experience with building relational database schemas, data modeling, and password hashing. I'm familiar with SQL, the computer science fundamentals (algorithms, data structures, and more), and industry tools (Visual Studio Code, Git, CL, GitHub, Jira, Confluence, and more).

In addition to my software engineering and problem solving skills, I also have strong skills in documentation and communication, knowledge of art and design, and a relentless drive to learn and grow. Of course I'm a continual self-taught developer, but I also possess a certificate in Software Engineering from Hackbright Academy's full-time software engineering program. Last year, I was awarded a scholarship by O'Reilly Learning Platform for unlimited access to the platform to learn about all things technology and business. Among other things, I'm poking around AI and GenAI in Enterprise. Previously, I earned undergraduate degrees in journalism and mass communication and studio art from Drake University and the San Francisco Art Institute (SFAI).

Thanks for visiting my website, https://luccawang.dev, that I hand built and maintain!
